Welcome to OneMessageBot documentation¶

О библиотеке¶

Данная библиотека реализует удобный фреймворк для создания ботов. Каждый бот созданный с помощью этой библиотеки работает в режиме "одно окно и кнопки". Фактически библиотека - это менеджер управления сообщением бота в телеграм, который сводит бота к представлению в виде одного окна и клавиатуры.

Функции:

MongoDB (встроенная поддержка)

SQLite (встроенная поддержка)

JDBC указание Connection

Создание многостраничных клавиатур

Многоязычность с возможностью указания собственных переводов

Обработка событий в методах согласно ViewMessageImpl

Аннотация @ViewMessageListener для регистрации обработчиков событий

Поддержка уведомлений

Отправка текстовых сообщений

Отправка изображений с указанием ссылки на изображение, java.io.File, byte[]

Отправка видео с указанием ссылки на видео, java.io.File, byte[]

Отправка аудио с указанием ссылки на аудио, java.io.File, byte[]

Отправка документов с указанием ссылки на документ, java.io.File, byte[]

Отправка Reply Keyboard

Установка¶

Maven:

<dependency>
    <groupId>ru.zoommax</groupId>
    <artifactId>OneMessageBot</artifactId>
    <version>1.0</version>
</dependency>

Gradle:

implementation 'ru.zoommax:OneMessageBot:1.0'
